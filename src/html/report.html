O que é percentil?

      
Percentil é uma medida estatística que indica a posição relativa de um valor dentro de um conjunto de dados
        ordenados. Por exemplo, o 90º percentil (P90) representa o valor abaixo do qual 90% dos dados se encontram. No
        contexto de latência, isso significa que 90% das requisições foram mais rápidas ou tão rápidas quanto o valor do
        P90,
        e os 10% restantes foram mais lentos.
